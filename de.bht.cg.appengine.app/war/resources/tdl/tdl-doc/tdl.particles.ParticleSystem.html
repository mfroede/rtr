Class: ParticleSystem

tdl.particles. ParticleSystem

new ParticleSystem

Creates a particle system. You only need one of these to run multiple emitters of different types of particles.
Parameters:
Name Type Description
gl !WebGLRenderingContext The WebGLRenderingContext into which the particles will be rendered.
opt_clock !function(): number A function that returns the number of seconds elapsed. The "time base" does not matter; it is corrected for internally in the particle system. If not supplied, wall clock time defined by the JavaScript Date API will be used.
opt_randomFunction !function(): number A function that returns a random number between 0.0 and 1.0. This allows you to pass in a pseudo random function if you need particles that are reproducible.

createTrail

Creates a Trail particle emitter. You can use this for jet exhaust, etc...
Parameters:
Name Type Description
parent !o3d.Transform Transform to put emitter on.
maxParticles number Maximum number of particles to appear at once.
parameters !tdl.particles.ParticleSpec The parameters used to generate particles.
opt_texture !o3d.Texture The texture to use for the particles. If you don't supply a texture a default is provided.
opt_perParticleParamSetter !function(number, !tdl.particles.ParticleSpec): void A function that is called for each particle to allow it's parameters to be adjusted per particle. The number is the index of the particle being created, in other words, if numParticles is 20 this value will be 0 to 19. The ParticleSpec is a spec for this particular particle. You can set any per particle value before returning.
opt_clock !function(): number A function to be the clock for the emitter.
