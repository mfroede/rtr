<static> exportSymbol

Exposes an unobfuscated global namespace path for the given object. Note that fields of the exported object *will* be obfuscated, unless they are exported in turn via this function or tdl.exportProperty.

Also handy for making public items that are defined in anonymous closures. ex. tdl.exportSymbol('Foo', Foo); ex. tdl.exportSymbol('public.path.Foo.staticFunction', Foo.staticFunction); public.path.Foo.staticFunction(); ex. tdl.exportSymbol('public.path.Foo.prototype.myMethod', Foo.prototype.myMethod); new public.path.Foo().myMethod();

Parameters:
Name Type Description
publicPath string Unobfuscated name to export.
object Object Object the name should point to.
opt_objectToExportTo Object The object to add the path to; default is |tdl.global|.
